![Vercel](https://vercelbadge.vercel.app/api/ciiiii/notion-blog?style=for-the-badge)

## Quickstart
### Develop
```bash
# 1. install dependencies
yarn
# 2. start server
yarn run dev
```
### Production
```bash
# 1. install dependencies
yarn
# 2. build svelte
yarn build
# 3. start server
node api/server.js
```
